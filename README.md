# CodeBranch - Frontend

**CodeBranch** est une application web permettant aux développeurs de centraliser leurs liens importants (GitHub, portfolio, blog, etc.) dans un espace simple, élégant et personnalisable.

Cette partie du projet correspond au **frontend**, développé avec **React**, **Vite**, **TailwindCSS**, et **TypeScript**.

---

## Stack technique

- [React 18](https://reactjs.dev/)
- [Vite](https://vitejs.dev/)
- [TailwindCSS](https://tailwindcss.com/)
- TypeScript
- [React Router v6](https://reactrouter.com/)
- SCSS Modules (avec `@apply` de Tailwind)
- Structure modulaire et scalable

---
